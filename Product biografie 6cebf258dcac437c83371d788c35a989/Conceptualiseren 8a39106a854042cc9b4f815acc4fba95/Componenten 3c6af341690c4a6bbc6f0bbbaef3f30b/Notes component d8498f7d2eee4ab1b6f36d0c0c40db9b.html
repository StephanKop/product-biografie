Notes component

Het Notes component is het 7e component van het dashboard. Dit component heeft dezelfde functie als een prikbord met stickynotes. De verwachting is dus dat de dev’s eens in de zoveel tijd er even op zullen kijken of notes eraan toevoegen. Hierdoor is het prima om dit component op een andere pagina te plaatsen.

Goals

  • Developers weten wat er speelt binnen het team

Technische uitdaging

De Notes moeten aangemaakt, aangepast en verwijderd kunnen worden.

Deze requests moeten dan verwerkt worden op de backend en doorgestuurd naar de database.

Technisch

Technische flow

Notes moeten aangemaakt, aangepast en verwijderd kunnen worden.

Op de backend heb ik verschillende routes aangemaakt. Een post route, een put route en een delete route. 

@Get()
  getNotes(@Param() params) {
    return this.notesService.getNotes();
  }
@Delete(':id')
  remove(@Param('id') id: string) {
    return this.notesService.remove(+id);
  }
@Post()
  async createDeadline(@Body() note: CreateNoteDto):Promise<CreateNoteDto> {
    return await this.notesService.createNotes(note)
  }

Notities renderen met .map() functie

useEffect functie

Om zonder page reload elke keer nieuwe data te laten zien maak ik gebruik van de useEffect functie. Deze luisterd naar een wijziging in een state waarna hij de interface zonder page reload herlaad.

useEffect(() => {
        fetchData();
        let interval = setInterval(() => fetchData(), (1000 * 10))
        return () => clearInterval(interval)
    }, [isLoaded]);

Design

Eerste ontwerp

Voor de notities heb ik besloten om het card type ontwerp aan te houden zoals ook gebruikt wordt in het member component. Hierdoor krijgt de gehele applicatie een duidelijke ontwerp style. Op een notitie worden de titel, beschrijving en datum van toevoegen getoond.

Iteratie 1

Aan de hand van feedback van stefan en ron heb ik besloten om mijn notes component kwa front-end om te gooien. Het was namelijk vooral onduidelijk wat de pagina precies inhield. Hierdoor heb ik besloten om de notities meer op sticky notes te laten lijken.

Ook heb ik een verwijder optie toegevoegd aangezien ik die in de eerste versie vergeten was. Deze hoort uiteraard wel bij de mogelijkheden.
