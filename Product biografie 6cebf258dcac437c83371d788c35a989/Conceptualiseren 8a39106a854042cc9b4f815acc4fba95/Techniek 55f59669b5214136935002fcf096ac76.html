Server logging van remote

https://linuxhint.com/send_linux_logs_remote_server/

In bovenstaand artikel wordt gebruik gemaakt van het linux programma rsyslog. Dit programma kan server logs versturen naar een remote server. Dit lijkt in eerste instantie een goede oplossing om server logging op te halen. Echter is het wel belangrijk om uit te zoeken of er een logging server opgezet moet worden of dat dit gewoon naar een ftp server gestuurd kan worden waar ik het dan weer ophaal met mijn applicatie.

Healthchecks

https://docs.nestjs.com/recipes/terminus

Uit de documentatie blijkt dat NestJS beschikt over dedicated healthcheck systemen. Deze zit niet standaard ingebouwd maar is te installeren via npm en is een originele package van NestJS. De healthcheck geeft de mogelijkheid om de volgende onderdelen te checken.

  • DNSHealthIndicator
  • TypeOrmHealthIndicator
  • MongooseHealthIndicator
  • MicroserviceHealthIndicator
  • GRPCHealthIndicator
  • MemoryHealthIndicator
  • DiskHealthIndicator
