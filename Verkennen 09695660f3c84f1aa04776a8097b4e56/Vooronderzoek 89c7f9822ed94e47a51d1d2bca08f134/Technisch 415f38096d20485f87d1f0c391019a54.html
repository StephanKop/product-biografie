  • Backend development

    Waarom is er een backend nodig?

    De backend voor deze applicatie is voornamelijk voor het afhandelen en opslaan van de data. Deze data zal dan weer opgehaald kunnen worden door de front-end en daar op de gewenste manier getoond worden. Door deze informatie op te slaan kan er een statistieken geschiedenis opgebouwd worden.

    Deze vraag heb ik ook aan bob voorgelegd. Zijn antwoord was:

    We hebben een backend nodig om alle data gestructureerd af te handelen voor het een database in gaat. Deadline, member, project data zal allemaal opgeslagen moeten worden om op dezelfde manier getoond te worden aan iedereen
